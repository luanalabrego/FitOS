# FitOS

> Seu Sistema Operacional Fitness

FitOS e um aplicativo mobile-first para gestao de alimentacao e treinos. Centralizamos tudo para voce: treinos, dietas e acompanhamento de resultados.

## Objetivos Suportados

- **Ganho de Massa Muscular** - Treinos de hipertrofia e dietas hipercaloricas
- **Perda de Peso** - Deficit calorico inteligente com treinos otimizados
- **Condicionamento Fisico** - Melhore sua resistencia e saude geral

## Tech Stack

- **Next.js 14** - React framework com App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Estilizacao utility-first
- **Lucide React** - Icones modernos

## Como Rodar

```bash
# Instalar dependencias
npm install

# Rodar em desenvolvimento
npm run dev

# Build para producao
npm run build

# Rodar producao
npm start
```

## Estrutura

```
src/
  app/           # App Router (pages e layouts)
  components/    # Componentes reutilizaveis
  lib/           # Utilitarios e helpers
public/          # Assets estaticos
```
